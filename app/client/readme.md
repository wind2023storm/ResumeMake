# latexresu.me server
This contains the code for the client side app for latexresu.me.

### Install dependencies
```
npm install
```

### Starting the dev server (localhost:3000)
```
npm start
```

### Create production bundle
```
npm run build
```

### Lint code
```
npm run lint
```

### Fix lint issues
```
npm run fix
```

### Format code
```
npm run format
```

### Run tests
```
npm test
```
