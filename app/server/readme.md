# server-boilerplate

A custom boilerplate for my server-side projects. The boilerplate includes:

* Koa 2
* Koa-Router
* Koa-Static
* Standard linting

## Installation
```
yarn
```

or

```
npm install
```

## Usage

Start the server:

```
npm start
```

Watch changes on server:

```
npm run watch
```
