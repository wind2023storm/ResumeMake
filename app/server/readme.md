# latexresu.me server
This contains the code for the api backend for latexresu.me.

### Install dependencies
```
npm install
```

### Starting the server (localhost:3001)
```
npm start
```

### Starting the server in watch mode (localhost:3001)
```
npm run watch
```

### Lint code
```
npm run lint
```

### Fix lint issues
```
npm run fix
```

### Format code
```
npm run format
```

### Run tests
```
npm test
```
